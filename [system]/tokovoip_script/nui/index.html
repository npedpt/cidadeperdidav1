TokoVOIP
TokoVOIP é a nossa substituição de VOIP no jogo, usando um TS Plugin personalizado.

Isso nos dá muito mais possibilidades, que atualmente permitem coisas como:
  • Melhor qualidade de voz
  • Sistema de rádio
  • Bate-papo por proximidade exclusivo para cada jogador
  • Ligações telefônicas
  • Cliques em rádio
  • Efeitos de rádio

TokoVOIP tem que ser instalado e usado para jogar nas nossas cidades.

Se não puder ingressar no canal do servidor apropriado, entre no canal no teamspeak, para que um membro da equipe de suporte possa ajudá-lo.

You can find more information and download links at: http://Cidadeperdida.tk
